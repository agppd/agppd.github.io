# agppd团队's blog

这是我团队的博客，基于 [Jekyll](https://jekyllrb.com/) 和默认主题 [minima](https://github.com/jekyll/minima) 搭建。

## 本地预览

1. 安装 Ruby 和 Bundler
2. 克隆本仓库
3. 运行 `bundle install`
4. 运行 `bundle exec jekyll serve`
5. 访问 `http://127.0.0.1:4000`

## 在线访问

访问 [agppd团队 的博客](https://agppd.github.io/)

# agppd Team's Blog
This is our team's blog, built with [Jekyll](https://jekyllrb.com/) and the default theme [minima](https://github.com/jekyll/minima).

## Local Preview
1. Install Ruby and Bundler
2. Clone this repository
3. Run `bundle install`
4. Run `bundle exec jekyll serve`
5. Visit `http://127.0.0.1:4000`

## Online Access
Visit [agppd's Blog](https://agppd.github.io/)
