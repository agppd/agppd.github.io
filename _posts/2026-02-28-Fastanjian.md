###  第一部分：通用操作（两者大致相同）

这些快捷键在日常使用中最为高频，且跨平台通用性最强。

| 功能 | Windows | Linux (大多数桌面环境) |
| :--- | :--- | :--- |
| **复制** | `Ctrl + C` | `Ctrl + C` |
| **剪切** | `Ctrl + X` | `Ctrl + X` |
| **粘贴** | `Ctrl + V` | `Ctrl + V` |
| **撤销** | `Ctrl + Z` | `Ctrl + Z` |
| **重做** | `Ctrl + Y` | `Ctrl + Y` / `Ctrl + Shift + Z` |
| **保存** | `Ctrl + S` | `Ctrl + S` |
| **全选** | `Ctrl + A` | `Ctrl + A` |
| **查找** | `Ctrl + F` | `Ctrl + F` |
| **切换应用** | `Alt + Tab` | `Alt + Tab` |
| **强制退出程序** | `Ctrl + Shift + Esc` (打开任务管理器) | `Ctrl + Alt + Esc` (直接杀死窗口) |
| **锁屏** | `Win + L` | `Ctrl + Alt + Del` (或 `Super + L`，取决于设置) |

### 第二部分：Windows 系统专用指南

#### 1. Win 键（也称 Windows 键）组合
`Win` 键是 Windows 的精华所在。

- **`Win`**：打开/关闭开始菜单。
- **`Win + D`**：显示桌面（最小化所有窗口，再按一次恢复）。
- **`Win + E`**：打开文件资源管理器（我的电脑）。
- **`Win + L`**：**锁定电脑**（最常用的离开保护快捷键）。
- **`Win + M`**：最小化所有窗口。
- **`Win + Shift + M`**：还原最小化的窗口。
- **`Win + Tab`**：打开任务视图（时间线/虚拟桌面切换）。
- **`Win + Ctrl + D`**：创建新的虚拟桌面。
- **`Win + Ctrl + F4`**：关闭当前虚拟桌面。
- **`Win + Ctrl + ←/→`**：在虚拟桌面之间切换。
- **`Win + 数字键(1,2,3...)`**：打开或切换到任务栏上对应位置的程序。
- **`Win + 方向键`**：将当前窗口贴靠到屏幕左/右/最大化/最小化。
- **`Win + R`**：打开“运行”对话框。
- **`Win + S` (或 `Win + Q`)**：打开搜索。
- **`Win + I`**：打开设置。
- **`Win + X`**：打开快捷菜单（设备管理器、磁盘管理等）。

#### 2. 文件资源管理器专用
- **`Alt + D`**：定位到地址栏（方便输入路径）。
- **`Ctrl + N`**：打开新的窗口。
- **`Ctrl + W`**：关闭当前窗口。
- **`Ctrl + Shift + N`**：**新建文件夹**。
- **`F2`**：重命名选中的文件/文件夹。
- **`F5`**：刷新。
- **`Alt + ↑`**：向上移动一级目录。
- **`Alt + ←/→`**：前进/后退浏览文件夹。

#### 3. 实用截图与录屏
- **`PrtSc (Print Screen)`**：截取全屏并复制到剪贴板（需粘贴到画图工具才能保存）。
- **`Win + PrtSc`**：截取全屏并自动保存为图片（通常保存在“图片”>“屏幕截图”文件夹）。
- **`Alt + PrtSc`**：截取当前活动窗口并复制到剪贴板。
- **`Win + Shift + S`**：打开截图工具（可自由选区、窗口或全屏截图）。

###  第三部分：Linux 系统专用指南

*注意：Linux 有不同的桌面环境（如 GNOME, KDE Plasma, XFCE 等），快捷键可能略有差异。以下以最常见的 **GNOME** 和通用习惯为例。*

- **`Super` 键**（通常就是键盘上的 Windows 键）：打开活动概览（相当于 macOS 的 Mission Control，显示所有窗口和工作区）。

#### 1. 窗口与工作区管理
- **`Alt + Tab`**：切换应用程序。
- **`Alt + \``（波浪号，Tab上方）**：在同一个应用程序的多个窗口间切换。
- **`Super + A`**：显示所有应用程序菜单。
- **`Super + ↑/↓`**：最大化/恢复当前窗口（或最小化）。
- **`Ctrl + Alt + ↑/↓`**：切换到上一个/下一个工作区。
- **`Ctrl + Alt + Shift + ↑/↓`**：将当前窗口移动到上一个/下一个工作区。
- **`Super + H`**：隐藏（最小化）当前窗口。
- **`Alt + F4`**：关闭当前窗口。
- **`Alt + F10`**：切换窗口最大化状态。
- **`Alt + Space`**：打开窗口操作菜单（移动、调整大小等）。

#### 2. 终端 Terminal 
Linux 用户离不开终端，掌握这些能极大提升效率。

- **`Ctrl + Alt + T`**：打开一个新终端窗口（大部分发行版默认）。
- **`Ctrl + Shift + C`**：**复制**（终端里通常需要加 Shift）。
- **`Ctrl + Shift + V`**：**粘贴**（终端里通常需要加 Shift）。
- **`Ctrl + Shift + T`**：在终端中打开一个新标签页。
- **`Ctrl + PageUp/PageDown`**：在终端标签页之间切换。
- **`Ctrl + Shift + W`**：关闭当前终端标签页。
- **`Ctrl + Shift + Q`**：关闭整个终端窗口。
- **`Ctrl + L`**：清屏（相当于输入 `clear` 命令）。
- **`Ctrl + C`**：终止当前正在运行的命令。
- **`Tab`**：自动补全命令或文件名（极其重要）。
- **`↑ / ↓`**：浏览之前执行过的命令历史。

#### 3. 系统管理
- **`Ctrl + Alt + Del`**：在大多数 Linux 桌面版中打开注销/重启/关机对话框（或锁屏）。
- **`Ctrl + Alt + F1~F6`**：切换到虚拟控制台（TTY，纯命令行界面）。
- **`Ctrl + Alt + F7`**（或 F1）：切换回图形界面（取决于发行版配置）。
- **`PrtSc`**：打开截图工具（GNOME 下）。
- **`Alt + PrtSc`**：截取当前窗口。
- **`Shift + PrtSc`**：截取屏幕区域。

###  第四部分：实用技巧与自定义

#### 1. 创建自定义快捷方式

**Windows：**
1.  右键点击桌面上的程序快捷方式（或任何可执行文件）。
2.  选择 **属性**。
3.  在 **快捷方式** 选项卡中，点击 **快捷键** 输入框。
4.  按下你想设置的组合键（例如 `Ctrl + Alt + N`）。
5.  点击确定。以后按这个组合键就能启动该程序。

**Linux (以 GNOME 为例)：**
1.  打开 **设置**。
2.  进入 **键盘** (Keyboard) 或 **键盘快捷键** (Keyboard Shortcuts)。
3.  滑动到底部，点击 **+** 号添加自定义快捷键。
4.  输入名称和命令，然后点击 **设置快捷键** 并按下组合键。

#### 2. 为什么有些快捷键冲突了？
- 如果某个全局快捷键不起作用，通常是因为你当前正在使用的程序（如 IDE、游戏、虚拟机）占用了这个组合键。
- 在 Linux 中，你可以通过上述设置路径，找到冲突的快捷键进行修改或禁用。
