## 1. Lua 简介

——by Deepseek

Lua 是一种轻量、小巧、可嵌入的脚本语言，由巴西里约热内卢天主教大学（PUC-Rio）的 Roberto Ierusalimschy、Waldemar Celes 和 Luiz Henrique de Figueiredo 开发。其名字源自葡萄牙语“lua”，意为“月亮”。

**主要特点**：
- **简洁高效**：语法简单，易于学习，执行速度快。
- **可嵌入**：可以轻松集成到 C/C++ 等语言中，作为扩展脚本。
- **动态类型**：变量无类型，值有类型。
- **自动内存管理**：采用垃圾回收机制。
- **多范式**：支持过程式、函数式、面向对象和基于原型（通过元表）编程。
- **跨平台**：几乎可在所有操作系统上运行。

**应用场景**：
- 游戏开发（如 World of Warcraft、Angry Birds 的脚本）
- 嵌入式系统（如路由器、摄像头配置）
- 独立应用程序扩展脚本
- 数据描述（配置文件）

## 2. 环境搭建与第一个程序
### 2.1 安装 Lua
- **Windows**：可以从 [Lua 官网](https://www.lua.org/download.html) 下载二进制文件，或使用 LuaDist、LuaForWindows 等集成环境。
- **Linux**：通过包管理器安装（如 `sudo apt install lua5.3`）。
- **macOS**：使用 Homebrew：`brew install lua`。

安装后，在终端输入 `lua -v` 验证安装。

### 2.2 运行第一个程序
创建一个文件 `hello.lua`，内容如下：
```lua
print("Hello, Lua!")
```
在终端执行：
```bash
lua hello.lua
```
输出：
```
Hello, Lua!
```

也可以直接进入交互模式：
```bash
$ lua
Lua 5.3.5  Copyright (C) 1994-2018 Lua.org, PUC-Rio
> print("hello")
hello
>
```

## 3. 基础语法与数据类型
### 3.1 注释
- **单行注释**：`-- 这是注释`
- **多行注释**：
```lua
--[[
这是多行注释
可以跨越多行
]]
```

### 3.2 变量
Lua 是动态类型语言，变量无需声明类型，直接赋值即可。默认情况下变量是全局的，除非使用 `local` 显式声明为局部变量。
```lua
x = 10          -- 全局变量
local y = "hi"  -- 局部变量
```
访问未初始化的变量得到 `nil`。

### 3.3 数据类型
Lua 有 8 种基本类型：
- `nil`      —— 空值（表示无效值）
- `boolean`  —— 布尔值（`true` 或 `false`）
- `number`   —— 数字（双精度浮点数，也可表示整数）
- `string`   —— 字符串（不可变，支持单引号、双引号、长括号 `[[...]]`）
- `table`    —— 表（关联数组，可作数组、字典、对象等）
- `function` —— 函数（第一类值，可赋值给变量、作为参数传递）
- `thread`   —— 线程（协程）
- `userdata` —— 用户数据（用于 C 语言扩展）

使用 `type()` 函数可获取值的类型：
```lua
print(type(nil))        --> nil
print(type(true))       --> boolean
print(type(10))         --> number
print(type("hello"))    --> string
print(type({}))         --> table
print(type(print))      --> function
```

### 3.4 字符串
字符串可以用单引号、双引号或长括号定义。长括号支持多行字符串：
```lua
a = "hello"
b = 'world'
c = [[
多行
字符串
]]
```
字符串连接使用 `..` 运算符：
```lua
print("Hello " .. "World")  --> Hello World
```
获取字符串长度：`#` 运算符或 `string.len` 函数：
```lua
print(#"hello")  --> 5
```

## 4. 表达式与运算符
### 4.1 算术运算符
- `+` 加法
- `-` 减法
- `*` 乘法
- `/` 除法
- `^` 乘方
- `%` 取模
- `-` 负号

示例：
```lua
print(10 + 2)   --> 12
print(10 % 3)   --> 1
print(2 ^ 3)    --> 8.0
```

### 4.2 关系运算符
- `==` 等于
- `~=` 不等于
- `>` 大于
- `<` 小于
- `>=` 大于等于
- `<=` 小于等于

比较结果返回布尔值。

### 4.3 逻辑运算符
- `and` 逻辑与
- `or`  逻辑或
- `not` 逻辑非

逻辑运算符短路求值，返回参与运算的值（不一定是布尔值）：
```lua
print(4 and 5)       --> 5
print(nil or "default") --> default
```

### 4.4 其他运算符
- `..` 字符串连接
- `#` 取长度（适用于字符串和表）

## 5. 控制结构
### 5.1 条件语句
```lua
if condition then
    -- 代码块
elseif other_condition then
    -- 代码块
else
    -- 代码块
end
```
示例：
```lua
x = 10
if x > 0 then
    print("positive")
elseif x == 0 then
    print("zero")
else
    print("negative")
end
```

### 5.2 循环
#### while 循环
```lua
while condition do
    -- 循环体
end
```
示例：
```lua
i = 1
while i <= 5 do
    print(i)
    i = i + 1
end
```

#### repeat...until 循环
类似 do...while，至少执行一次，直到条件为真时退出。
```lua
repeat
    -- 循环体
until condition
```
示例：
```lua
i = 1
repeat
    print(i)
    i = i + 1
until i > 5
```

#### for 循环
- **数值 for**：
```lua
for var = start, end, step do
    -- 循环体
end
```
step 默认为 1，可省略。
示例：
```lua
for i = 1, 10, 2 do
    print(i)   -- 1,3,5,7,9
end
```

- **泛型 for**：常用于遍历表。
```lua
for key, value in pairs(table) do
    -- 循环体
end
```
示例：
```lua
t = {a=1, b=2, c=3}
for k, v in pairs(t) do
    print(k, v)
end
```

### 5.3 break 和 return
- `break`：跳出循环。
- `return`：从函数返回值。在块的最后一行可以省略分号。

注意：`break` 和 `return` 只能作为块的最后一条语句，如果要在块中间使用，需用 `do return end` 包裹。

## 6. 函数
### 6.1 定义函数
函数使用 `function` 关键字定义，可以有参数和返回值。
```lua
function add(a, b)
    return a + b
end
```
也可以将函数赋值给变量：
```lua
add = function(a, b) return a + b end
```

### 6.2 多返回值
函数可以返回多个值，用逗号分隔。
```lua
function swap(a, b)
    return b, a
end
x, y = swap(1, 2)   -- x=2, y=1
```

### 6.3 可变参数
使用 `...` 表示可变参数，在函数内部用 `{...}` 收集参数表，或用 `select` 函数访问。
```lua
function sum(...)
    local s = 0
    for i, v in ipairs({...}) do
        s = s + v
    end
    return s
end
print(sum(1,2,3,4))  --> 10
```

### 6.4 闭包（Closure）
Lua 支持词法作用域，函数可以访问其外部函数的局部变量。当内部函数引用外部局部变量时，就形成了闭包。
```lua
function counter()
    local count = 0
    return function()
        count = count + 1
        return count
    end
end
c1 = counter()
print(c1())  --> 1
print(c1())  --> 2
```

### 6.5 匿名函数
函数是值，可以匿名定义并直接使用：
```lua
t = {1, 2, 3}
table.sort(t, function(a, b) return a > b end)  -- 降序排序
```

## 7. 表（Table）
表是 Lua 中唯一的数据结构，可以表示数组、字典、集合、对象等。它是关联数组，索引可以是数字、字符串或其他类型的值（除 nil）。

### 7.1 创建表
```lua
-- 空表
a = {}
-- 初始化
b = {10, 20, 30}                 -- 隐式索引从1开始
c = {x=10, y=20}                  -- 字符串键
d = {[1]=10, [2]=20, ["name"]="Lua"}
```

### 7.2 访问与修改
```lua
t = {name="Lua", version=5.3}
print(t.name)          --> Lua
print(t["version"])    --> 5.3
t.author = "PUC-Rio"   -- 新增字段
t.name = "Lua 5.3"     -- 修改
t[1] = 100             -- 数字键
```

### 7.3 表的遍历
- `pairs`：遍历所有键值对（顺序不确定）。
- `ipairs`：遍历连续的数字键（从1开始直到遇到 nil）。
```lua
t = {10, 20, 30, name="Lua"}
for k, v in pairs(t) do
    print(k, v)
end
-- 输出（顺序可能不同）：
-- 1   10
-- 2   20
-- 3   30
-- name Lua

for i, v in ipairs(t) do
    print(i, v)  -- 1 10, 2 20, 3 30 （name不会被遍历）
end
```

### 7.4 表的标准库函数
- `table.insert(list, [pos,] value)`：插入元素。
- `table.remove(list, [pos])`：删除元素并返回。
- `table.sort(list, [comp])`：排序。
- `table.concat(list, [sep, i, j])`：连接字符串元素。
- `table.unpack(list)`：将表元素展开为多个值。

示例：
```lua
t = {3, 1, 4, 1, 5}
table.sort(t)          --> {1,1,3,4,5}
table.insert(t, 2, 9)  --> {1,9,1,3,4,5}
print(table.concat(t, ", "))  --> 1, 9, 1, 3, 4, 5
```

### 7.5 表的长度
使用 `#` 操作符获取表的**序列**长度，即从1开始的连续整数键的数量。对于非序列表，结果不可靠。
```lua
t = {1, 2, 3, nil, 5}
print(#t)  --> 3 （因为键4为nil，停止计数）
```

## 8. 元表与元方法
元表（metatable）允许我们改变表的行为。每个表可以有一个元表，其中定义了当表执行特定操作时调用的元方法。

### 8.1 设置元表
```lua
t = {}
mt = {}
setmetatable(t, mt)   -- 设置t的元表为mt
getmetatable(t)       --> 返回元表
```

### 8.2 常用元方法
- `__index`：当访问表中不存在的键时调用。
- `__newindex`：当给表中不存在的键赋值时调用。
- `__add`、`__sub`、`__mul`、`__div`、`__mod`、`__pow`、`__unm`：算术运算。
- `__concat`：连接运算（`..`）。
- `__eq`、`__lt`、`__le`：关系运算。
- `__call`：当表作为函数调用时触发。
- `__tostring`：当表被转换为字符串时（如 `print(t)`）触发。

示例：使用 `__index` 实现默认值
```lua
t = {a=1, b=2}
mt = {
    __index = function(table, key)
        return "default"
    end
}
setmetatable(t, mt)
print(t.a)      --> 1
print(t.c)      --> default
```

使用 `__add` 重载加法：
```lua
t1 = {10, 20}
t2 = {30, 40}
mt = {
    __add = function(a, b)
        local result = {}
        for i = 1, #a do
            result[i] = a[i] + b[i]
        end
        return result
    end
}
setmetatable(t1, mt)
setmetatable(t2, mt)
t3 = t1 + t2
for _, v in ipairs(t3) do print(v) end  --> 40, 60
```

## 9. 面向对象编程
Lua 本身没有类，但可以利用表和元表模拟面向对象。常用的方法是使用原型模式，将对象的元表指向一个“类”表，并利用 `__index` 实现继承和方法查找。

### 9.1 简单类
```lua
-- 定义一个类（其实是一个表）
Animal = {name = "Animal"}

function Animal:new(name)
    local obj = {name = name or self.name}
    setmetatable(obj, self)
    self.__index = self
    return obj
end

function Animal:speak()
    print(self.name .. " makes a sound.")
end

-- 创建实例
a = Animal:new("Dog")
a:speak()  --> Dog makes a sound.
```

### 9.2 继承
```lua
Dog = Animal:new()   -- Dog继承Animal
function Dog:speak()
    print(self.name .. " barks.")
end

d = Dog:new("Buddy")
d:speak()   --> Buddy barks.
```
这里 `Dog:new()` 创建了一个新表，其元表指向 `Animal`，所以 `Dog` 拥有 `Animal` 的 `new` 方法，同时我们可以覆盖 `speak` 方法。

## 10. 协程（Coroutine）
协程是协作式多任务处理，可以暂停和恢复执行。Lua 通过 `coroutine` 库提供支持。

### 10.1 创建协程
```lua
co = coroutine.create(function()
    for i = 1, 3 do
        print("co", i)
        coroutine.yield()   -- 暂停，返回控制权
    end
end)
```

### 10.2 恢复执行
```lua
coroutine.resume(co)  --> co 1
coroutine.resume(co)  --> co 2
coroutine.resume(co)  --> co 3
coroutine.resume(co)  -- 无输出，协程已结束，返回false和错误信息
```

### 10.3 协程状态
```lua
print(coroutine.status(co))  --> suspended（运行中为 running，结束后为 dead）
```

### 10.4 传递数据
`resume` 可以传递参数给协程，`yield` 可以返回值。
```lua
co = coroutine.create(function(a, b)
    print("start", a, b)
    local r = coroutine.yield(a + b)   -- 返回 a+b，并接收下次resume传入的参数
    print("resumed with", r)
    return a * b
end)

print(coroutine.resume(co, 3, 4))  --> true, start 3 4, 7 （resume返回true和yield的参数）
print(coroutine.resume(co, 10))    --> true, resumed with 10, 12 （最终返回）
```
`resume` 的第一个返回值表示协程是否成功执行，后续值是 `yield` 或 `return` 的参数。

## 11. 模块与包
Lua 通过 `require` 函数加载模块，避免全局命名空间污染。模块通常返回一个表，包含所有公开的函数和变量。

### 11.1 编写模块
创建一个文件 `mymodule.lua`：
```lua
local M = {}

local function private_helper()
    print("private")
end

function M.public_func()
    print("public function")
    private_helper()
end

return M
```

### 11.2 使用模块
```lua
local mymodule = require("mymodule")
mymodule.public_func()
```
`require` 会在 `package.path` 指定的路径中查找文件，并且只加载一次，后续 `require` 返回缓存的结果。

### 11.3 包路径
`package.path` 是 Lua 查找 lua 文件的路径模板，默认包含当前目录和标准库路径。可以自定义：
```lua
package.path = package.path .. ";/usr/local/share/lua/?.lua"
```

## 12. 错误处理与调试
### 12.1 错误抛出
使用 `error(message [, level])` 抛出错误。
```lua
if n < 0 then
    error("n must be non-negative")
end
```

### 12.2 断言
`assert(condition, message)` 在条件为假时抛出错误。
```lua
local file = assert(io.open("test.txt", "r"))
```

### 12.3 保护调用
- `pcall`（protected call）捕获错误，返回状态和结果。
```lua
local ok, result = pcall(function()
    error("oops")
end)
if not ok then
    print("error:", result)
end
```
- `xpcall` 类似，但可指定错误处理函数（通常用于获取调用栈）。

### 12.4 调试库
Lua 提供 `debug` 库，可以获取函数信息、设置钩子、查看局部变量等。
```lua
debug.traceback()  -- 获取当前调用栈信息
```

## 13. 标准库详解
### 13.1 字符串库（string）
- `string.len(s)` 或 `#s`
- `string.sub(s, i, j)` 提取子串
- `string.format(format, ...)` 格式化（类似 C 的 printf）
- `string.find(s, pattern [, init])` 查找模式
- `string.gsub(s, pattern, repl [, n])` 全局替换
- `string.match(s, pattern)` 匹配并返回捕获
- `string.gmatch(s, pattern)` 生成迭代器

示例：
```lua
print(string.format("pi = %.2f", math.pi))  --> pi = 3.14
print(string.find("hello world", "world"))  --> 7 11
```

### 13.2 表库（table）
前面已介绍过 `table.insert`、`table.remove`、`table.sort`、`table.concat` 等。还有 `table.move`（移动元素）、`table.pack`（将参数打包成表）等。

### 13.3 数学库（math）
常用函数：`math.abs`、`math.sin`、`math.cos`、`math.sqrt`、`math.random`、`math.max`、`math.min`、`math.floor`、`math.ceil`。
```lua
math.randomseed(os.time())   -- 设置随机种子
print(math.random(1, 100))   -- 1到100随机整数
```

### 13.4 I/O 库
提供文件读写操作。
- `io.open(filename, mode)` 打开文件，返回文件句柄。
- 模式："r" 读，"w" 写，"a" 追加，"b" 二进制。
- 常用方法：`file:read("*all")`、`file:lines()`、`file:write(...)`、`file:close()`。

示例：
```lua
local f = assert(io.open("test.txt", "w"))
f:write("Hello\n")
f:close()

local f = io.open("test.txt", "r")
for line in f:lines() do
    print(line)
end
f:close()
```

### 13.5 操作系统库（os）
- `os.clock()` 返回 CPU 时间
- `os.date(format)` 格式化日期
- `os.difftime(t2, t1)` 时间差
- `os.execute(command)` 执行系统命令
- `os.exit([code])` 退出程序
- `os.getenv(varname)` 获取环境变量
- `os.remove(filename)` 删除文件
- `os.rename(oldname, newname)` 重命名
- `os.time([table])` 获取时间戳

示例：
```lua
print(os.date("%Y-%m-%d %H:%M:%S"))  --> 2025-01-01 12:00:00
```

### 13.6 调试库（debug）
常用函数：
- `debug.getinfo([thread,] f [, what])` 获取函数信息
- `debug.getlocal([thread,] f, local)` 获取局部变量
- `debug.setlocal([thread,] level, local, value)`
- `debug.traceback([thread,] [message [, level]])` 生成调用栈

## 14. 高级主题
### 14.1 垃圾回收
Lua 采用自动内存管理，使用增量标记-清除垃圾回收器。可以通过 `collectgarbage` 函数控制。
- `collectgarbage("collect")` 强制执行一次完整回收
- `collectgarbage("count")` 返回当前 Lua 使用的内存（KB）
- `collectgarbage("step", size)` 执行一步回收

### 14.2 弱表（Weak Table）
弱表允许键或值是弱引用，不会被垃圾回收器计入可达性。通过元表的 `__mode` 字段设置。
- `__mode = "k"` 键弱引用
- `__mode = "v"` 值弱引用
- `__mode = "kv"` 两者都是弱引用

示例：缓存对象，当对象不再被其他地方引用时自动从表中移除。
```lua
cache = {}
setmetatable(cache, {__mode = "v"})  -- 值是弱引用

obj = {}
cache[1] = obj   -- 放入缓存
obj = nil        -- 原引用释放
collectgarbage() -- 触发回收
print(cache[1])  --> nil （对象已被回收）
```

### 14.3 C API 与嵌入
Lua 设计为可嵌入，C 程序可以通过 Lua C API 与 Lua 交互。主要包括：
- 创建 Lua 状态机：`lua_State *L = luaL_newstate();`
- 加载库：`luaL_openlibs(L);`
- 执行 Lua 代码：`luaL_dostring(L, "print('hello')");`
- 在 C 中注册函数供 Lua 调用。
- 从 Lua 栈中获取参数和返回值。

示例片段：
```c
#include <lua.h>
#include <lauxlib.h>
#include <lualib.h>

int main() {
    lua_State *L = luaL_newstate();
    luaL_openlibs(L);
    luaL_dostring(L, "print('Hello from C!')");
    lua_close(L);
    return 0;
}
```
编译时需要链接 Lua 库。

## 15. 实战示例
### 15.1 简单的游戏脚本
假设我们有一个游戏，需要读取配置文件并控制角色行为。以下是一个简单的脚本，定义了一个角色和它的移动函数。

**角色模块**（character.lua）：
```lua
local Character = {}

function Character:new(name, x, y)
    local obj = {name = name, x = x, y = y}
    setmetatable(obj, self)
    self.__index = self
    return obj
end

function Character:move(dx, dy)
    self.x = self.x + dx
    self.y = self.y + dy
    print(self.name .. " moved to (" .. self.x .. ", " .. self.y .. ")")
end

return Character
```

**主程序**（main.lua）：
```lua
local Character = require("character")

local player = Character:new("Hero", 0, 0)
player:move(10, 5)
player:move(-2, 3)
```

输出：
```
Hero moved to (10, 5)
Hero moved to (8, 8)
```

### 15.2 配置文件处理
用 Lua 表作为配置文件，易于解析。

**config.lua**：
```lua
return {
    window = {width = 800, height = 600, title = "My Game"},
    sound = {volume = 0.8, mute = false},
    player = {name = "Player1", level = 1}
}
```

**加载配置**：
```lua
local config = require("config")
print(config.window.width)  --> 800
```
