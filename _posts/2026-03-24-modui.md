## 题意

输入一个长度为 $n$ 的序列，值域为 $[1,k]$，$m$ 组查询，每次查询 $[L,R]$ 区间内每种数字出现次数的平方和。$1 \le n,m,k \le 10^5$。

## 莫队算法

莫队是一种巧妙的离线算法，适用于可以 $O(1)$ 从 $[l,r]$ 转移到 $[l \pm 1,r]$ 或 $[l,r \pm 1]$ 的问题。核心解题步骤是：

1. 将序列分成长度为 $B$ 的块（通常情况下，$B=\sqrt n$）。

2. 把所有查询按照左端点所在的块作为第一关键字，右端点的值作为第二关键字排序。

3. 使用两个指针不断向每个查询区间移动，并且每次移动 $O(1)$ 计算贡献。

## 本题做法

设当前区间 $[L,R]$ 中某数字 $x$ 出现次数为 $c$。

- 添加一个数字 $x$ 时，其出现次数由 $c$ 变为 $c+1$，平方和变化量为 $(c+1)^2 - c^2 = 2c+1$。

- 删除一个数字 $x$ 时，其出现次数由 $c$ 变为 $c-1$，平方和变化量为 $(c-1)^2 - c^2 = -2c+1$。

因此我们可以在移动指针时 $O(1)$ 更新答案。

## 正确性证明

莫队算法通过离线，在每次移动指针时剔除或添加贡献，正确性显然。

## 时间复杂度证明

设块大小为 $B$，共有 $\lceil n/B \rceil$ 个块。

- **左指针移动**

在同一块内，最劣的情况是每次询问的左端点都是该块左右端点交替，单次移动 $B$ 次，总移动次数为 $m \cdot B$。

在不同块内，从一个块跳到他相邻右边的一个块，最多会跳 $\lceil n/B \rceil$ 次，且每次块间跳转最多移动 $B$ 步，所以共会跳 $\lceil n/B \rceil \times B=n$ 次。

综上，左指针移动次数最多为 $m \cdot B + n$ 次，单次移动 $O(1)$，总复杂度为 $O(m \cdot B + n)$。

- **右指针移动**

在同一块内，由于以右端点大小为第二关键字排序，所以右端点单调，右指针单次移动次数显然不超过 $n$，共 $\lceil n/B \rceil$ 个块，共移动 $n\cdot \lceil n/B \rceil$。

在不同块内，完成一组询问后，右指针会停留在该组最后一个询问的右端点处。接下来处理下一组询问时，右指针需要移动到下一组第一个询问的右端点。这个跳转的距离最坏为 $n$（如从 $1$ 跳到 $n$），共有 $\lceil n/B \rceil$ 块，最多跳 $\lceil n/B \rceil-1$ 次，因此共会跳 $n \cdot \lceil n/B \rceil$ 次。

综上，右指针移动次数最多次数 $n \cdot \lceil n/B \rceil = \frac{n^2}{B}$，单次移动 $O(1)$，总时间复杂度 $O(\frac{n^2}{B})$。

- **总复杂度**

将做右指针时间复杂度相加，得到总复杂度 $O(m \cdot B + n+\frac{n^2}{B})=O((n+m) \sqrt n)$。

因此无论询问如何分布，总复杂度上界均为 $O((n+m)\sqrt{n})$。

## 代码实现

**分块**

设 $block$ 为块长，则 $block = \sqrt n$，记录 $B_i$ 为 $i$ 所在块的编号。

**排序**

先按左端点块号排序，左端点位于同一块内按右端点从小到大排序。

这里可以进一步做优化，具体地，当左端点所在的块号为奇数时，块内右端点按升序排列；当左端点所在的块号为偶数时，块内右端点按降序排列。

这样做的好处是，相邻两个块的右端点移动方向相反，使得右指针在块间跳转时的移动距离显著减小。例如：奇数块内右端点从小到大，处理完后右指针位于较大的位置，下一个偶数块内右端点从大到小，第一个询问的右端点也较大，因此右指针只需稍微左移即可，避免了从远端跳回。

**指针移动**

对于大部分莫队题目，都和如下模板一样，其中 $l,r$ 表示当前查询区间，$L,R$ 表示当前左右指针。

```cpp
while(L<l){
    //剔除L的贡献
    L++;
}
while(R>r){
    //剔除R的贡献
    R--;
}
while(L>l){
    L--;
    //添加L的贡献
}
while(R<r){
    R++;
    //添加R的贡献
}
```

对于本题，上文已经说明如何计算新增减贡献，在这里不过多赘述。

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,m,k;
int res;
int block;
int a[100005];
int B[100005];
int cnt[100005];
long long ans[100005];
struct node{
    int l,r,id;
}q[100005];
bool cmp(node x,node y){
    if(B[x.l]!=B[y.l]) return B[x.l]<B[y.l];
    if(B[x.l]&1) return x.r<y.r;
    return x.r>y.r;
}
int main(){
    ios::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    cin>>n>>m>>k;
    block=sqrt(n);
    for(int i=1;i<=n;i++){
        cin>>a[i];
        if(i%block==1) B[i]=++res;
        else B[i]=B[i-1];
    }
    for(int i=1;i<=m;i++){
        cin>>q[i].l>>q[i].r;
        q[i].id=i;
    }
    sort(q+1,q+1+m,cmp);
    int L=1,R=0;
    long long nwans=0;
    for(int i=1;i<=m;i++){
        int l=q[i].l;
        int r=q[i].r;
        while(L<l){
            nwans-=2*cnt[a[L]]-1;
            cnt[a[L]]--;
            L++;
        }
        while(R>r){
            nwans-=2*cnt[a[R]]-1;
            cnt[a[R]]--;
            R--;
        }
        while(L>l){
            L--;
            cnt[a[L]]++;
            nwans+=2*cnt[a[L]]-1;
        }
        while(R<r){
            R++;
            cnt[a[R]]++;
            nwans+=2*cnt[a[R]]-1;
        }
        ans[q[i].id]=nwans;
    }
    for(int i=1;i<=m;i++){
        cout<<ans[i]<<'\n';
    }
    return 0;
}
```

## 总结

莫队算法是一种优雅的离线处理方法，通过分块与排序将区间查询的复杂度控制在 $O((n+m) \sqrt{n})$ 以内，特别适合那些可以 $O(1)$ 转移答案的问题。本文从算法思想、正确性证明到代码实现进行了详细讲解，希望能帮助读者掌握这一经典算法。
