## **1. 文档与文本文件**

| 后缀名 | 类型 | 简介/对比 |
| :--- | :--- | :--- |
| **.txt** | 纯文本 | 无格式，体积最小，兼容性最高。 |
| **.rtf** | 富文本 | 有基础格式（字体/颜色），比TXT丰富，比DOC兼容性好。 |
| **.doc** | Word文档（旧） | Office 97-2003专用，二进制格式。 |
| **.docx** | Word文档（新） | 基于XML，比DOC**文件更小、结构更稳定**。 |
| **.xls** | Excel表格（旧） | 早期Excel电子表格。 |
| **.xlsx** | Excel表格（新） | 基于XML，**支持更多行和列**，文件更小。 |
| **.ppt** | PPT演示稿（旧） | 早期PowerPoint演示文稿。 |
| **.pptx** | PPT演示稿（新） | 基于XML，**功能更丰富、动画效果更流畅**。 |
| **.pdf** | 便携式文档 | **跨平台布局固定**，不可随意编辑，适合打印/传阅。 |
| **.odt** | 开放文档 | 开源软件（LibreOffice）标准，与DOCX功能类似。 |


## **2. 图像文件**

| 后缀名 | 类型 | 简介/对比 |
| :--- | :--- | :--- |
| **.bmp** | 位图 | **无压缩**，图像质量最高，**文件体积巨大**。 |
| **.jpg /.jpeg** | 联合图像专家组 | **有损压缩**，体积小，色彩丰富，**网络照片主流格式**。 |
| **.png** | 便携式网络图形 | **无损压缩**，支持透明背景，压缩比高于GIF，**网页设计首选**。 |
| **.gif** | 图形交换格式 | 支持256色，**支持动态图**，色彩不如JPG丰富。 |
| **.psd** | Photoshop源文件 | **保存图层/通道/编辑信息**，方便修改，体积大。 |
| **.tif /.tiff** | 标签图像 | 高质量位图，支持多种压缩，**广泛应用于印刷行业**。 |
| **.svg** | 可缩放矢量图形 | **矢量图无限放大不失真**，网页矢量图标准。 |
| **.cdr** | CorelDRAW | 矢量绘图软件专用，常用于广告设计。 |


## **3. 音频文件**

| 后缀名 | 类型 | 简介/对比 |
| :--- | :--- | :--- |
| **.wav** | 波形文件 | **未经压缩**，音质完美，**文件体积巨大**（CD音质）。 |
| **.mp3** | 动态影像专家压缩音频 | **有损压缩**，舍弃人耳不敏感频率，**体积小、流行度高**。 |
| **.flac** | 自由无损音频压缩编码 | **无损压缩**，体积比WAV小，**音质完全还原**（发烧友首选）。 |
| **.aac** | 高级音频编码 | 有损压缩，**比MP3压缩效率更高、音质更好**。 |
| **.wma** | Windows Media Audio | 微软有损格式，兼容Windows系统。 |
| **.mid /.midi** | 乐器数字接口 | **不记录声音波形，只记录音符指令**，**文件极小**（电子乐谱）。 |


## **4. 视频文件**

| 后缀名 | 类型 | 简介/对比 |
| :--- | :--- | :--- |
| **.avi** | 音频视频交错格式 | 微软老牌格式，兼容性好，**压缩率低，文件偏大**。 |
| **.mp4** | MPEG-4 Part 14 | **目前最通用的标准**，压缩率高，画质好，流媒体主流。 |
| **.mov** | QuickTime电影 | 苹果公司格式，**常用于专业影视编辑**。 |
| **.mkv** | Matroska视频 | **开源封装容器**，可容纳多字幕/音轨，高清电影常见。 |
| **.flv** | Flash Video | 早期网络视频主流，已逐渐被MP4淘汰。 |
| **.rmvb** | RealMedia可变比特率 | 早期网络流媒体，高压缩率，画质一般。 |


## **5. 压缩与镜像文件**

| 后缀名 | 类型 | 简介/对比 |
| :--- | :--- | :--- |
| **.zip** | ZIP压缩 | **开放、通用**，几乎所有平台原生支持。 |
| **.rar** | RAR压缩 | 比ZIP**压缩比更高**，有**恢复记录功能**，需专用软件。 |
| **.7z** | 7-Zip压缩 | **极高压缩率**（尤其适合大量小文件），开源。 |
| **.tar.gz /.tar.bz2** | 打包压缩 | Linux/Unix常用组合：tar打包，gzip/bzip2压缩。 |
| **.iso** | 光盘镜像 | 光盘/磁盘的**完整映像**，符合ISO 9660标准。 |


## **6. 系统与可执行文件**

| 后缀名 | 类型 | 简介/对比 |
| :--- | :--- | :--- |
| **.exe** | 可执行文件 | Windows系统中**可直接运行的程序**。 |
| **.msi** | Windows安装包 | 微软安装程序格式，双击运行安装软件。 |
| **.dll** | 动态链接库 | **被多个程序共享的代码/数据**，不可直接运行。 |
| **.sys** | 系统文件 | 包含硬件驱动程序或系统配置。 |
| **.bat** | 批处理文件 | 包含一系列命令的脚本，**自动化任务**。 |
| **.ini** | 初始化配置文件 | 保存软件设置参数，**可用记事本编辑**。 |


## **7. 编程与开发文件**

| 后缀名 | 类型 | 简介/对比 |
| :--- | :--- | :--- |
| **.c** | C语言源码 | C语言源代码文件。 |
| **.cpp /.cc** | C++语言源码 | C++语言源代码文件。 |
| **.java** | Java源码 | Java源代码文件。 |
| **.py** | Python源码 | Python脚本文件。 |
| **.js** | JavaScript | 网页脚本语言文件。 |
| **.html /.htm** | 超文本标记语言 | **网页文件**，浏览器打开。 |
| **.php** | PHP脚本 | 服务器端脚本文件。 |
| **.h** | 头文件 | C/C++语言的头文件，用于声明函数/变量。 |
| **.class** | Java字节码 | Java编译后文件，可在JVM上运行。 |
