## 一、 什么是 GNU？

**GNU** 是一个自由操作系统项目，名称源于递归缩写 **“GNU is Not Unix”**（GNU不是Unix）。它由理查德·斯托曼（Richard Stallman）于1983年发起，旨在创建一个完全自由、兼容 Unix 的操作系统。

GNU 项目的核心在于 **自由软件精神**，强调“自由”指代言论自由（Free Speech），而非免费（Free Beer）。它定义了四项基本自由：
- **自由0**：无论出于何种目的，按意愿运行软件。
- **自由1**：研究并修改软件以适应自身需求（前提是能获取源代码）。
- **自由2**：自由分发拷贝以帮助他人。
- **自由3**：自由分发修改后的版本，使社区受益。

为实现这一目标，GNU 发布了 **GPL（通用公共许可证）**，通过“著佐权”（Copyleft）机制，确保所有衍生作品同样保持自由。

> **注**：我们今天广泛使用的 **Linux** 操作系统，实际上是 **Linux 内核** 与 **GNU 工具集** 的结合，更准确的称呼是 **GNU/Linux 系统**。

[GNU官方网址](https://www.gnu.org/home.zh-cn.html)

## 二、 GNU 常用软件

GNU 项目提供了从底层系统到上层应用的丰富软件。以下按类别介绍其中最具代表性、广泛使用的工具。

### 1. 核心基础工具

这些是 GNU/Linux 系统的基础，也是所有类 Unix 系统的标配。

| 软件 | 功能描述 | 特点与用途 |
| :--- | :--- | :--- |
| **Bash** | GNU Bourne-Again SHell | 最流行的命令行解释器。具备强大的命令历史、Tab 补全和脚本编程能力，是系统管理与开发的核心交互界面。 |
| **GNU Coreutils** | 核心工具集 | 包含 `ls`、`cat`、`cp`、`mv`、`rm`、`grep`、`find` 等数百个基础命令。它们是系统操作的基石，高效且稳定。 |
| **GRUB** | GRand Unified Bootloader | 主流的多系统引导程序。允许用户在开机时选择启动哪个操作系统，支持多种文件系统和内核格式。 |

### 2. 文本编辑与开发

GNU 在软件开发领域贡献了业界公认的标杆工具。

| 软件 | 功能描述 | 特点与用途 |
| :--- | :--- | :--- |
| **Emacs** | 可扩展、可定制的文本编辑器 | 被称为“伪装成编辑器的操作系统”。通过 Elisp 语言可扩展为邮件客户端、文件管理器、IDE 等，全键盘操作哲学极大提升效率。 |
| **GCC** | GNU 编译器套件 | 业界标准的编译器，支持 C、C++、Fortran、Ada、Go 等多种语言。几乎所有开源项目和 Linux 内核均由 GCC 编译，以优秀的优化能力和跨平台支持著称。 |
| **GDB** | GNU 调试器 | 强大的程序调试工具。支持断点设置、内存查看、反汇编等，帮助开发者精准排查程序崩溃和逻辑错误。 |
| **Make** | 自动化构建工具 | 通过 `Makefile` 定义编译规则，自动管理大型项目的编译依赖，极大简化了构建流程。 |

### 3. 办公与图形

GNU 项目也提供了功能强大的图形应用，是许多用户拥抱自由软件的直接理由。

| 软件 | 功能描述 | 特点与用途 |
| :--- | :--- | :--- |
| **GIMP** | GNU 图像处理程序 | 免费开源的 Photoshop 替代品。功能强大，支持插件扩展，适用于照片修图、图形设计、图像格式转换等专业工作。 |
| **Inkscape** | 矢量图形编辑器 | 专业的 SVG 矢量图绘制工具，对标 Adobe Illustrator，广泛用于图标设计、流程图制作和插画创作。 |
| **Gnash** | SWF 影片播放器 | GNU 开发的 Flash 播放器，致力于在自由环境中流畅播放 SWF 格式内容。 |
| **Gnumeric** | 电子表格软件 | 轻量级但功能完备的电子表格工具，处理速度快，公式兼容性高，是办公场景中的优秀选择。 |

## 一、包管理器安装

### 常用安装

| 软件 | Debian/Ubuntu | Fedora | Arch Linux |
| :--- | :--- | :--- | :--- |
| **Bash** | 已预装 | 已预装 | 已预装 |
| **GCC** | `sudo apt install build-essential` | `sudo dnf install gcc` | `sudo pacman -S gcc` |
| **Emacs** | `sudo apt install emacs` | `sudo dnf install emacs` | `sudo pacman -S emacs` |
| **GIMP** | `sudo apt install gimp` | `sudo dnf install gimp` | `sudo pacman -S gimp` |
| **Inkscape** | `sudo apt install inkscape` | `sudo dnf install inkscape` | `sudo pacman -S inkscape` |

> **说明**：Bash和GNU Coreutils（ls、cp、grep等）在几乎所有Linux发行版中都是默认预装的，无需手动安装。


## 二、跨发行版安装方案

如果需要**最新版本**或使用的是**非主流发行版**，可以采用以下方式：

### 1. Flatpak（推荐）

适合追求最新版本且希望环境隔离的用户。

```bash
# 安装Flatpak（以Ubuntu为例）
sudo apt install flatpak

# 添加Flathub仓库
flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo

# 安装GIMP
flatpak install flathub org.gimp.GIMP

# 启动GIMP
flatpak run org.gimp.GIMP
```

### 2. AppImage（无需安装）

适合不想污染系统环境或临时使用的场景。

```bash
# 下载AppImage文件后赋予执行权限
chmod +x gimp-*.AppImage

# 直接运行
./gimp-*.AppImage
```


## 三、源码编译安装

### GCC 源码编译示例

```bash
# 下载源码（以GCC 13.2.0为例）
wget https://ftp.gnu.org/gnu/gcc/gcc-13.2.0/gcc-13.2.0.tar.gz
tar -xzf gcc-13.2.0.tar.gz
cd gcc-13.2.0

# 下载依赖
./contrib/download_prerequisites

# 配置、编译、安装
./configure --prefix=/usr/local/gcc-13.2.0 --enable-languages=c,c++
make -j$(nproc)
sudo make install
```

### GRUB 源码编译示例

```bash
# 安装编译依赖
sudo apt install bison gcc make flex

# 下载并编译GRUB 2.02
wget https://alpha.gnu.org/gnu/grub/grub-2.02~rc1.tar.gz
tar -xzf grub-2.02~rc1.tar.gz
cd grub-2.02~rc1
mkdir build && cd build
../configure
make && sudo make install
```


## 四、特定工具的部署说明

### 1. Bash 配置加载

Bash本身无需安装，但可以通过`source`命令加载自定义配置：

```bash
# 加载自定义配置
source ~/.my_config

# 加载环境变量文件
source app_env.sh
```

> `source`命令在当前shell中执行脚本，使变量和函数修改立即生效，与直接运行脚本（会启动子shell）不同。

### 2. Emacs 基本配置

安装后建议创建配置文件`~/.emacs.d/init.el`进行个性化设置：

```elisp
;; 设置默认字体
(set-face-attribute 'default nil :family "Fira Code" :height 120)

;; 关闭启动画面
(setq inhibit-startup-screen t)

;; 显示行号
(global-display-line-numbers-mode t)
```

### 3. GIMP 启动方式

- **包管理器安装**：终端输入`gimp`或通过应用菜单启动
- **Flatpak安装**：`flatpak run org.gimp.GIMP`
- **AppImage**：直接双击文件运行

### 4. Inkscape 启动

安装完成后，终端输入`inkscape`即可启动。

### 5. Guix 包管理器（GNU高级工具）

Guix是GNU项目的函数式包管理器，支持可重现的环境构建：

```bash
# 首次使用需初始化
source /opt/ohpc/pub/guix-start.sh
guix pull

# 安装软件包
guix install emacs python@3 gcc-toolchain@8

# 创建独立profile
guix install -p ~/my-profile gcc-toolchain@8 lapack

# 进入特定环境
guix shell -p ~/my-profile
```
