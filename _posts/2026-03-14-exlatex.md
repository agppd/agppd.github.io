## 1. 通用基础宏包

无论撰写哪个学科的文档，以下宏包都建议加载，它们提供了基础功能与字体、页面、图表等支持。

| 宏包 | 用途 | 示例 |
|------|------|------|
| `inputenc` | 指定源文件编码（如 UTF-8） | `\usepackage[utf8]{inputenc}` |
| `fontenc` | 指定输出编码（如 T1） | `\usepackage[T1]{fontenc}` |
| `geometry` | 设置页面边距 | `\usepackage[a4paper,margin=2.5cm]{geometry}` |
| `babel` | 多语言支持（如英文、中文） | `\usepackage[english]{babel}` |
| `amsmath`, `amssymb` | 数学公式增强 | `\usepackage{amsmath,amssymb}` |
| `graphicx` | 插入图片 | `\includegraphics[width=0.5\textwidth]{figure.png}` |
| `hyperref` | 生成超链接、书签 | `\usepackage{hyperref}` |
| `caption`, `subcaption` | 自定义图表标题，支持子图 | `\usepackage{caption,subcaption}` |
| `listings` / `minted` | 代码高亮 | `\usepackage{listings}` 或 `\usepackage{minted}` |
| `tikz` | 绘制矢量图形 | `\usepackage{tikz}` |

## 2. 数学与计算机科学

### 2.1 数学

- **`amsmath`**：基础数学环境（`equation`, `align`, `gather` 等）。
- **`amsthm`**：定理环境（`\newtheorem`）。
- **`mathtools`**：`amsmath` 的扩展，提供更多数学工具。
- **`siunitx`**：科学单位与数值排版。
- **`physics`**：物理学常用符号（导数、括号、向量等），但数学中也可用。

```latex
\documentclass{article}
\usepackage{amsmath, amsthm, mathtools, siunitx, physics}
\theoremstyle{definition}
\newtheorem{theorem}{Theorem}[section]
\newtheorem{definition}[theorem]{Definition}

\begin{document}
\section{示例}
\begin{definition}[极限]
  设函数 $f(x)$ 在 $x_0$ 的某邻域内有定义，若存在常数 $A$，使得对任意 $\epsilon > 0$，
  存在 $\delta > 0$，当 $0 < |x - x_0| < \delta$ 时，有 $|f(x) - A| < \epsilon$，
  则称 $A$ 为 $f(x)$ 当 $x \to x_0$ 的极限，记为
  \[
  \lim_{x \to x_0} f(x) = A.
  \]
\end{definition}

\begin{theorem}[夹逼准则]
  若在 $x_0$ 的某去心邻域内，$g(x) \le f(x) \le h(x)$，且
  $\lim_{x \to x_0} g(x) = \lim_{x \to x_0} h(x) = A$，则
  \[
  \lim_{x \to x_0} f(x) = A.
  \]
\end{theorem}

物理单位示例：$\SI{3.14}{\micro\meter}$，导数 $\dv{f}{x}$，偏导 $\pdv{f}{x,y}$。
\end{document}
```

### 2.2 计算机科学

- **`listings`**：代码块排版，支持多种语言。
- **`minted`**：依赖 Pygments，语法高亮更美观。
- **`algorithm2e`** / **`algorithmicx`**：算法伪代码排版。

```latex
\documentclass{article}
\usepackage{listings}
\usepackage{xcolor}
\lstset{
  language=Python,
  basicstyle=\ttfamily\small,
  keywordstyle=\color{blue},
  commentstyle=\color{gray},
  stringstyle=\color{red},
  showstringspaces=false,
  numbers=left,
  numberstyle=\tiny,
  frame=single
}

\usepackage[ruled,vlined]{algorithm2e}

\begin{document}
\section{代码示例}
\lstinputlisting[caption={Python 示例}]{example.py}

\begin{algorithm}[H]
  \SetAlgoLined
  \KwData{一个整数 $n$}
  \KwResult{斐波那契数列前 $n$ 项}
  \BlankLine
  $F[0] \leftarrow 0$\;
  $F[1] \leftarrow 1$\;
  \For{$i \leftarrow 2$ \KwTo $n-1$}{
    $F[i] \leftarrow F[i-1] + F[i-2]$\;
  }
  \Return $F$\;
  \caption{斐波那契数列}
\end{algorithm}
\end{document}
```

## 3. 物理学

- **`physics`**：提供 `\dd`, `\dv`, `\pdv`, `\va`, `\vb` 等快捷命令。
- **`siunitx`**：规范单位和数值。
- **`circuitikz`**：绘制电路图。
- **`feynmp`** / **`tikz-feynman`**：费曼图。

```latex
\documentclass{article}
\usepackage{physics, siunitx, circuitikz}
\usepackage{tikz-feynman} % 需要 LuaLaTeX 或启用 external

\begin{document}
\section{物理公式}
薛定谔方程：
\[
i\hbar \pdv{\Psi}{t} = -\frac{\hbar^2}{2m} \laplacian{\Psi} + V\Psi.
\]

单位示例：$c = \SI{299792458}{\meter\per\second}$。

\section{电路图}
\begin{circuitikz}
  \draw (0,0) to[V=$V_{in}$] (0,2)
        to[R=$R$] (2,2)
        to[L=$L$] (4,2)
        to[C=$C$] (4,0) -- (0,0);
  \draw (2,2) -- (2,3) to[short,*-o] (2,3) node[right]{$V_{out}$};
\end{circuitikz}

\section{费曼图}
\begin{tikzpicture}
  \begin{feynman}
    \vertex (a) {\(e^-\)};
    \vertex [right=of a] (b);
    \vertex [right=of b] (c);
    \vertex [right=of c] (d) {\(e^-\)};
    \vertex [above=of b] (e) {\(\gamma\)};
    \diagram* {
      (a) -- [fermion] (b) -- [fermion] (c) -- [fermion] (d),
      (b) -- [photon] (e),
    };
  \end{feynman}
\end{tikzpicture}
\end{document}
```

## 4. 化学

- **`mhchem`**：化学方程式、化学式简便输入。
- **`chemfig`**：绘制有机化学结构式。

```latex
\documentclass{article}
\usepackage{mhchem, chemfig}

\begin{document}
\section{化学方程式}
水电解：\ce{2H2O -> 2H2 + O2}
\par
苯的硝化：\ce{C6H6 + HNO3 ->[H2SO4] C6H5NO2 + H2O}

\section{结构式}
\chemfig{*6(=-=-=-)} % 苯环
\par
\chemfig{CH_3-CH(-OH)-CH_3} % 异丙醇
\end{document}
```

## 5. 生物学

生物学通常使用通用宏包（如 `graphicx` 插入图片，`pgfplots` 绘制数据图），并无特定强制宏包。但以下宏包可能有用：

- **`pgfplots`**：数据绘图，适合生物统计图表。
- **`bioinfo`**：生物信息学专用，但较少用。
- **`tikz`** 绘制细胞、器官示意图。

```latex
\documentclass{article}
\usepackage{pgfplots}
\pgfplotsset{compat=1.18}
\usepackage{tikz}
\usetikzlibrary{shapes,arrows}

\begin{document}
\section{生长曲线}
\begin{tikzpicture}
  \begin{axis}[
    xlabel={时间 (h)},
    ylabel={细胞密度 (OD$_{600}$)},
    grid=major,
    legend pos=north west
  ]
    \addplot coordinates {(0,0.1) (2,0.3) (4,0.8) (6,1.5) (8,2.2) (10,2.5)};
    \addplot coordinates {(0,0.1) (2,0.2) (4,0.5) (6,1.0) (8,1.8) (10,2.3)};
    \legend{野生型, 突变型}
  \end{axis}
\end{tikzpicture}

\section{细胞示意图}
\begin{tikzpicture}
  \draw (0,0) circle (1);
  \draw (0,0) circle (0.6);
  \fill (0,0) circle (0.1);
  \node at (0,1.2) {细胞核};
  \node at (0,0.8) {细胞质};
\end{tikzpicture}
\end{document}
```

## 6. 工程学

工程学科常需要绘制图表、流程图、电路图、力学示意图等。

- **`tikz`** 及其库（`arrows`, `shapes`, `calc`, `patterns`）用于绘制机构简图。
- **`pgfplots`** 用于数据曲线。
- **`circuitikz`** 用于电路。
- **`algorithm2e`** 用于算法描述。
- **`siunitx`** 用于单位。

```latex
\documentclass{article}
\usepackage{tikz}
\usetikzlibrary{arrows,shapes,calc}
\usepackage{pgfplots}
\usepackage{siunitx}

\begin{document}
\section{力学简图}
\begin{tikzpicture}[>=stealth]
  \draw[thick] (-2,0) -- (2,0);
  \draw[fill=gray!30] (-0.5,0) rectangle (0.5,1);
  \draw[->,thick] (0,1) -- (0,2) node[above] {$F$};
  \draw[->,thick] (0,0) -- (0,-0.8) node[below] {$mg$};
  \draw[->,thick] (-0.8,0.5) -- (-1.5,0.5) node[left] {$f$};
  \node at (0,0.5) {m};
\end{tikzpicture}

\section{应力-应变曲线}
\begin{tikzpicture}
  \begin{axis}[
    xlabel={应变 $\varepsilon$},
    ylabel={应力 $\sigma$ (MPa)},
    grid=major,
    xmin=0, xmax=0.02,
    ymin=0, ymax=300
  ]
    \addplot[thick, blue] coordinates {
      (0,0) (0.001,100) (0.005,200) (0.01,250) (0.015,260) (0.02,250)
    };
  \end{axis}
\end{tikzpicture}
\end{document}
```

## 7. 语言学

语言学排版需要处理音标、例句、语法树等。

- **`tipa`** / **`fontspec`**（XeLaTeX）：国际音标。
- **`gb4e`** / **`linguex`** / **`expex`**：编号例句。
- **`tikz-qtree`** / **`forest`**：句法树。

```latex
\documentclass{article}
\usepackage{tipa}
\usepackage{gb4e}
\usepackage{tikz-qtree}

\begin{document}
\section{音标}
The pronunciation of ``ship'' is [\textshtipa].

\section{例句}
\begin{exe}
  \ex \label{ex:1} John saw Mary.
  \ex \label{ex:2} \gll 我 喜欢 书\\
          wo  xihuan shu\\
          \glt `I like books.'
\end{exe}

\section{句法树}
\begin{tikzpicture}
  \Tree [.S [.NP John ] [.VP [.V saw ] [.NP Mary ] ] ]
\end{tikzpicture}
\end{document}
```

## 8. 历史与哲学

这类人文学科通常不需要特殊宏包，但以下宏包有助于提升排版质量：

- **`biblatex`**：强大的参考文献管理。
- **`csquotes`**：智能引号。
- **`footmisc`** / **`manyfoot`**：自定义脚注。
- **`verse`**：诗歌排版。

```latex
\documentclass{article}
\usepackage[style=chicago]{biblatex}
\addbibresource{refs.bib}
\usepackage{csquotes}
\usepackage{verse}

\begin{document}
\section{引言}
\enquote{知之为知之，不知为不知，是知也。} ——《论语·为政》

\section{诗歌}
\begin{verse}
  关关雎鸠，在河之洲。\\
  窈窕淑女，君子好逑。
\end{verse}

\nocite{*}
\printbibliography
\end{document}
```

## 9. 音乐

- **`musixtex`**：五线谱排版。
- **`lilypond`** 更专业，但需外部程序；也可用 `lilypond-book` 嵌入 LaTeX。

```latex
\documentclass{article}
\usepackage{musixtex}

\begin{document}
\begin{music}
  \instrumentnumber{1}
  \setname1{Piano}
  \setstaffs1{2}
  \startextract
  \Notes \wh{c}\en
  \Notes \wh{e}\en
  \zendextract
\end{music}
\end{document}
```
