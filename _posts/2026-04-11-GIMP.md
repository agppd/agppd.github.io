## 一、前情提要

### 安装步骤

1. 访问GIMP官方网站（gimp.org）下载适合你操作系统的最新版本安装包。
2. 在Windows上，双击运行安装程序，按向导完成安装；在macOS上，可通过Homebrew命令 `brew install gimp`快速安装；在Linux上，使用 `sudo apt install gimp`（Ubuntu/Debian）即可。

### 界面布局解析

初次打开GIMP，你可能被满屏的面板吓到，其实只需熟悉三大核心区域即可：

- **左侧工具箱**：包含选择、画笔、变换等核心工具，鼠标悬停即可看到工具名称和快捷键。
- **中央画布区域**：图像编辑的主战场。
- **右侧面板**：图层面板（管理图层）、通道、路径、历史记录等。

建议新手花几分钟时间将鼠标悬停在每个工具上，阅读提示信息。熟悉工作区是掌握GIMP的第一步。

## 二、核心操作技巧

### 1. 图层与图层蒙版：非破坏性编辑的灵魂

图层是GIMP的核心概念。想象一下，每一层就像一张透明的玻璃纸，你在不同的玻璃纸上绘制不同的元素，叠在一起就形成完整的画面。这样做的好处是：修改其中一层不会影响其他层。

**新建图层**：点击“图层”菜单，选择“新建图层”，输入名称后确定即可。

**复制图层**：选中一个图层，点击“图层”菜单，选择“复制图层”，新图层会复制当前图层的所有内容。

**合并图层**：选中要合并的图层，点击菜单栏“图层”→“合并图层”，或用快捷键Ctrl+E。

**图层蒙版**是GIMP中最强大的功能之一，它允许你通过灰度信息控制图层哪些部分可见、哪些部分隐藏，整个过程完全可逆。

**如何添加图层蒙版**：

1. 在图层面板中选中目标图层。
2. 右键点击图层，选择“添加图层蒙版”。
3. 在弹出的对话框中选择初始状态。推荐选择“白色（全不透明）”，这样后续可以用黑色画笔“擦除”不需要的部分。

**如何使用蒙版进行局部调整**：

1. 点击选中图层右侧的蒙版缩略图（白色小方块）。
2. 选择画笔工具，设置前景色为黑色（用于隐藏）或白色（用于恢复）。
3. 在画面上涂抹：黑色让该区域透明（隐藏内容），白色让它重新显现。

> **进阶技巧**：如果想创建平滑的渐变过渡效果，可以选中蒙版缩略图后，使用渐变工具拖拽生成从黑到白的渐变。这特别适合做多张图像的融合过渡。

### 2. 抠图全攻略：七种去背方法

GIMP提供了丰富的选择工具来应对不同的抠图场景。根据图像特征选择合适的工具，能事半功倍：

| 工具               | 最佳使用场景                   |
| ------------------ | ------------------------------ |
| 矩形/椭圆选取      | 圆形、方形等规则形状           |
| 自由选取（套索）   | 粗略截取不规则形状             |
| 路径工具           | 曲线较多、边缘清晰的物体       |
| 模糊选取（魔术棒） | 背景单一或颜色对比明显         |
| 颜色选择           | 需要批量选取同色系区域         |
| 智能剪刀           | 自动追踪清晰边缘               |
| 前景抽取           | 只需粗略描绘即可分离背景与前景 |

**操作前必做的一步**：无论使用哪种抠图工具，务必先从“图层”菜单→“透明度”→“添加Alpha通道”，这样才能让删除的区域变成透明，而不是填充为白色或黑色。

**推荐路径：用路径工具实现精确抠图**

对于产品摄影、人像等需要精细边缘的场景，路径工具是最佳选择。它通过创建贝塞尔曲线路径，可以实现像素级的精确选择。

操作步骤：

1. 在工具箱中选择“路径工具”。
2. 沿着物体边缘点击放置锚点，在需要弯曲的地方点击并拖拽调整曲线弧度。
3. 闭合路径后，点击路径面板中的“路径转换为选区”按钮。
4. 按Delete删除背景，或添加图层蒙版来精细化边缘。

**选区边缘优化**：创建选区后，可以点击“选择”菜单中的“羽化”、“收缩”或“扩展”来微调选区边缘，让合成效果更加自然。对于人像抠图，3~5像素的羽化值通常能获得不错的效果。

### 3. 快速蒙版模式：用画笔“画”出选区

有时候，用路径工具逐点描边效率不够高。GIMP的快速蒙版模式允许你用画笔直接在图像上“画”出选区——红色区域表示被保护的（选中的部分）。点击工具箱底部的“快速蒙版”图标（或按Shift+Q），用画笔在需要选择的地方涂色，再按一次Shift+Q退出，涂色区域就会自动变成选区。这对于处理毛发、发丝等复杂边缘尤其高效。

### 4. 色彩校正与调色

**色彩平衡工具**

GIMP提供了强大的色彩管理功能，确保图像在不同设备上显示一致的颜色效果。

打开“颜色”菜单→“色彩平衡”，或按快捷键Ctrl+B调出面板。该工具包含三个核心参数区域：

- **色阶范围**：阴影（暗部）、中间调（大部分区域）、高光（亮部）。分别调整这三个范围可以实现精细的色彩控制。
- **颜色通道**：青-红、洋红-绿、黄-蓝三组滑块。向左滑动增加冷色，向右滑动增加暖色。

**实战技巧**：

- 校正偏色照片：如果图像整体偏绿，选择“中间调”，将洋红-绿滑块向洋红方向移动，同时观察实时预览。
- 营造复古效果：强化阴影部分的红色和高光部分的黄色，适当降低中间调的蓝色值。
- 局部调色：先复制图层并调整色彩平衡，然后用图层蒙版擦除不需要改变的区域。

**色彩管理基础设置**

对于追求专业色彩的用户，GIMP支持ICC配置文件管理。进入“编辑”→“首选项”→“色彩管理”，即可为图像配置合适的色彩空间。一般来说，sRGB适合网络图片和普通摄影，Adobe RGB适合专业摄影和印刷。

### 5. 裁剪与缩放

裁剪是图像处理中最基础也最常用的操作之一。选择裁剪工具（Shift+C），在画布上拖拽出想要保留的区域，按下Enter键即可完成裁剪。技巧提示：可以开启工具选项中的“三分法”辅助线，帮助你遵循摄影构图的基本原则。

如果需要调整图像尺寸，点击“图像”→“缩放图像”，输入目标宽度和高度即可。建议先裁剪后缩放，这样可以专注于最重要的画面区域。

## 三、综合实战案例：一张摄影后期的完整流程

现在，让我们通过一个完整的例子，把上面学到的知识点串联起来。假设你有一张在逆光下拍摄的人像照片，肤色偏暗、背景过亮，整体色彩有些偏冷。

### Step 1：复制图层，保留原始备份

打开照片后，在图层面板中右键点击背景图层，选择“复制图层”。将原图层重命名为“原始备份”，并将眼睛图标关闭（临时隐藏）。这样，你所有的操作都在副本上进行，随时可以参照原始效果。

### Step 2：使用色彩平衡调整肤色

选中复制出的图层，按Ctrl+B打开色彩平衡面板。人像肤色的偏色通常出现在中间调区域：将“黄-蓝”滑块向蓝色方向轻微拖动（消除黄色偏色），将“洋红-绿”滑块向洋红方向微调（增添血色）。勾选“预览”实时观察效果，避免过度调整。

### Step 3：使用曲线工具调整对比度

打开“颜色”→“曲线”（或按快捷键Ctrl+M）。在曲线上点击添加控制点：在亮部区域向上拉提亮高光，在暗部区域向下拉加深阴影，形成轻微的S形曲线，可以显著增强画面的通透感和立体感。

### Step 4：用图层蒙版进行局部调整

背景过亮、人物偏暗的问题还没解决。再复制一层刚才调整后的图层，点击“颜色”→“色阶”（Ctrl+L），将中间调滑块向左移动提亮整张图片（这会让背景更亮，但没关系）。然后给这个图层添加白色蒙版，选择画笔工具，设置前景色为黑色，在背景区域涂抹——你会发现背景变得更亮了，而人物区域因为被黑色蒙版遮盖，恢复到了之前的状态。这就是图层蒙版局部调整的魅力所在。

### Step 5：锐化与降噪

最后一步，合并所有可见图层（右键图层面板→“合并可见图层”），然后打开“滤镜”→“增强”→“锐化（USM）”，设置半径1.5像素、数量0.8左右，让人物眼睛和发丝更加清晰。如果画面有噪点，可以使用“滤镜”→“增强”→“降噪”进行适当处理。

### Step 6：导出成品

点击“文件”→“导出为”，选择JPEG或PNG格式。注意：在GIMP中，保存（Ctrl+S）保存的是带图层信息的XCF工程文件，而导出（Shift+Ctrl+E）才是生成最终图片文件。这个设计是为了让你始终保留可编辑的原始文件。
