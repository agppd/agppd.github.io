## 一、三种基本模式

Vim 通过模式切换实现不同操作。三种主要模式如下：

| 模式            | 功能            | 进入方式                          |
| ------------- | ------------- | ----------------------------- |
| 普通模式（Normal）  | 移动光标、删除、复制、粘贴 | 按 `Esc` 或 `Ctrl+[`            |
| 插入模式（Insert）  | 输入文本          | 按 `i`（光标前）、`a`（光标后）、`o`（下一行）等 |
| 命令模式（Command） | 保存、退出、搜索、替换   | 在普通模式下按 `:`                   |

操作流程一般为：普通模式 → 按 `i`/`a`/`o` 进入插入模式 → 输入文本 → 按 `Esc` 返回普通模式 → 按 `:` 进入命令模式执行保存/退出。

## 二、文件的打开、保存与退出

```bash
vim filename.txt   # 打开或创建文件
vim -R file        # 以只读方式打开
```

**Vim 内的保存与退出命令**（在命令模式下执行）：

| 命令           | 作用       |
| ------------ | -------- |
| `:w`         | 保存文件     |
| `:q`         | 退出       |
| `:wq` 或 `:x` | 保存并退出    |
| `:q!`        | 强制退出，不保存 |

## 三、普通模式下的光标移动

Vim 使用字母键 `h`、`j`、`k`、`l` 替代方向键移动光标：

```vim
h   ← 左移一格
j   ↓ 下移一行
k   ↑ 上移一行
l   → 右移一格
```

其他常用移动操作：

| 按键        | 作用                       |
| --------- | ------------------------ |
| `w` / `b` | 按单词前进/后退                 |
| `e`       | 跳到单词末尾                   |
| `0`       | 跳到行首                     |
| `^`       | 跳到行首第一个非空字符              |
| `$`       | 跳到行尾                     |
| `gg`      | 跳到文件第一行                  |
| `G`       | 跳到文件最后一行                 |
| `Ctrl+f`  | 向下翻一页                    |
| `Ctrl+b`  | 向上翻一页                    |
| `%`       | 跳转到匹配的括号（`{}`、`()`、`[]`） |

移动命令前可加数字表示重复次数，例如 `5j` 向下移动 5 行，`2w` 向前移动 2 个单词。

## 四、文本编辑操作

以下命令均在普通模式下执行。

### 删除（剪切）

| 命令    | 作用           |
| ----- | ------------ |
| `x`   | 删除光标下的字符     |
| `dd`  | 删除当前行        |
| `dw`  | 删除从光标位置开始的单词 |
| `d$`  | 删除从光标到行尾     |
| `d0`  | 删除从光标到行首     |
| `dgg` | 删除从光标到文件开头   |
| `dG`  | 删除从光标到文件末尾   |

数字重复：`3dd` 删除 3 行，`2dw` 删除 2 个单词。

### 撤销与重做

| 命令       | 作用         |
| -------- | ---------- |
| `u`      | 撤销上一步操作    |
| `Ctrl+r` | 重做（撤销撤销操作） |

### 复制与粘贴

| 命令   | 作用      |
| ---- | ------- |
| `yy` | 复制当前行   |
| `yw` | 复制一个单词  |
| `y$` | 复制到行尾   |
| `p`  | 粘贴到光标之后 |
| `P`  | 粘贴到光标之前 |

### 字符替换

| 命令  | 作用                  |
| --- | ------------------- |
| `r` | 替换光标下的单个字符（不进入插入模式） |
| `R` | 进入替换模式（覆盖式输入）       |

## 五、进入插入模式的多种方式

| 命令  | 作用         |
| --- | ---------- |
| `i` | 在光标前插入     |
| `I` | 在行首插入      |
| `a` | 在光标后插入     |
| `A` | 在行尾插入      |
| `o` | 在下方新建一行并插入 |
| `O` | 在上方新建一行并插入 |
| `s` | 删除当前字符并插入  |
| `S` | 删除当前行并插入   |

## 六、命令模式的操作

在普通模式下按 `:` 进入命令模式。

### 搜索

```vim
/pattern    向下搜索 pattern
?pattern    向上搜索 pattern
n           跳转到下一个匹配
N           跳转到上一个匹配
```

### 替换

```vim
:s/old/new/        替换当前行第一个匹配
:s/old/new/g       替换当前行所有匹配
:%s/old/new/g      替换整个文件所有匹配
:%s/old/new/gc     替换每个匹配前需要确认
```

### 显示行号

```vim
:set nu        显示行号
:set nonu      隐藏行号
:set number    同上
:set relativenumber  显示相对行号
```

## 七、进阶功能

### 分屏

```vim
:sp              水平分屏
:vsp             垂直分屏
Ctrl+w + hjkl    在分屏间移动光标
Ctrl+w + q       关闭当前分屏
```

### 宏录制

```vim
qa              开始录制到寄存器 a（q 开始录制，a 为寄存器名）
…执行若干操作
q               停止录制
@a              回放宏
@@              重复上一次宏
```

### 文件内跳转

```vim
Ctrl+o    跳转到之前的位置
Ctrl+i    跳转到之后的位置
```

### 多文件编辑

```vim
:e filename   在缓冲区中打开另一个文件
:bn           切换到下一个缓冲区
:bp           切换到上一个缓冲区
:bd           关闭当前缓冲区
```

## 八、配置文件 `.vimrc`

用户可在主目录下创建 `.vimrc` 文件以自定义 Vim 行为。示例配置：

```vim
" 基础设置
set number          " 显示行号
set relativenumber  " 相对行号
syntax on           " 语法高亮
set tabstop=4       " Tab 显示宽度为4空格
set expandtab       " 将 Tab 转换为空格
set autoindent      " 自动缩进
set showmatch       " 高亮匹配括号
set hlsearch        " 搜索高亮
set incsearch       " 实时搜索
set ignorecase      " 搜索忽略大小写
set smartcase       " 若搜索含大写字母则区分大小写

" 快捷键映射
let mapleader = ","
nnoremap <leader>w :w<CR>      " ,w 保存
nnoremap <leader>q :q<CR>      " ,q 退出
nnoremap <leader>h :nohlsearch<CR>  " 取消搜索高亮

" 插入模式下使用 jk 退出插入模式
inoremap jk <Esc>
```

修改配置后执行 `:source ~/.vimrc` 使其生效。

## 九、常见操作速查表

| 操作目的            | 命令序列                 |
| --------------- | -------------------- |
| 退出插入模式（不确定当前模式） | 多次按 `Esc`            |
| 撤销上一操作          | `u`                  |
| 重做              | `Ctrl+r`             |
| 删除整行            | `dd`                 |
| 删除单词            | `dw`                 |
| 复制行             | `yy`                 |
| 粘贴              | `p`                  |
| 保存文件            | `:w`                 |
| 不保存退出           | `:q!`                |
| 向下搜索 hello      | `/hello` 回车，按 `n` 继续 |
| 全局替换 foo 为 bar  | `:%s/foo/bar/g`      |
| 显示行号            | `:set nu`            |
| 跳转到第 42 行       | `42G` 或 `:42` 回车     |
