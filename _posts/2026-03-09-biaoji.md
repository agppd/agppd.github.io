## 引言
HTML、Markdown和LaTeX是三种常用的标记语言，它们各有侧重，但可以相互补充：

- **HTML（超文本标记语言）**：网页的骨架，用于构建网页内容的结构和呈现，是互联网的基础。
- **Markdown**：轻量级标记语言，设计目标是易读易写，常用于编写文档、博客、README文件等。
- **LaTeX**：基于TeX的排版系统，特别擅长处理复杂的科技文档、数学公式、学术论文。

本教程将分别介绍这三种语言的基础知识，并展示如何将它们结合使用，以满足多样化的文档编写需求。

## 第一部分：HTML教程

### 1.1 HTML简介
HTML（HyperText Markup Language）是用来描述网页的一种标记语言。它不是编程语言，而是一系列标签（tag）的集合，浏览器根据这些标签渲染页面内容。

### 1.2 HTML文档结构
一个基本的HTML文档如下：
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>我的第一个网页</title>
</head>
<body>
    <h1>欢迎来到我的网站</h1>
    <p>这是一个段落。</p>
</body>
</html>
```
- `<!DOCTYPE html>`：声明文档类型为HTML5。
- `<html>`：根元素，包含整个页面。
- `<head>`：头部，包含元数据（如编码、标题、样式链接等）。
- `<body>`：主体，包含所有可见内容。

### 1.3 常用标签
#### 标题与段落
```html
<h1>一级标题</h1>
<h2>二级标题</h2>
<p>这是一个段落。</p>
```

#### 文本格式化
```html
<strong>加粗</strong>
<em>斜体</em>
<u>下划线</u>
<del>删除线</del>
<br>换行
<hr>水平线
```

#### 链接与图片
```html
<a href="https://example.com">链接文本</a>
<img src="image.jpg" alt="描述文字" width="300">
```

#### 列表
```html
<!-- 无序列表 -->
<ul>
    <li>项目1</li>
    <li>项目2</li>
</ul>

<!-- 有序列表 -->
<ol>
    <li>第一步</li>
    <li>第二步</li>
</ol>
```

#### 表格
```html
<table border="1">
    <tr>
        <th>姓名</th>
        <th>年龄</th>
    </tr>
    <tr>
        <td>张三</td>
        <td>20</td>
    </tr>
</table>
```

### 1.4 表单与输入
```html
<form action="/submit" method="post">
    <label for="name">姓名：</label>
    <input type="text" id="name" name="name"><br>
    
    <label for="email">邮箱：</label>
    <input type="email" id="email" name="email"><br>
    
    <input type="submit" value="提交">
</form>
```

### 1.5 多媒体与嵌入
```html
<audio controls>
    <source src="music.mp3" type="audio/mpeg">
</audio>

<video controls width="400">
    <source src="video.mp4" type="video/mp4">
</video>

<!-- 嵌入其他网页 -->
<iframe src="https://example.com" width="600" height="400"></iframe>
```

### 1.6 HTML5新特性
- 语义化标签：`<header>`、`<nav>`、`<section>`、`<article>`、`<footer>`等。
- 表单新类型：`email`、`url`、`number`、`date`等。
- 绘图：`<canvas>`、`<svg>`。
- 本地存储：`localStorage`和`sessionStorage`。

## 第二部分：Markdown教程

### 2.1 Markdown简介
Markdown是一种轻量级标记语言，由John Gruber于2004年创建。它使用简单的符号（如`#`、`*`、`-`）来格式化文本，最终可转换为HTML。其设计目标是易读易写，并且可以方便地转换为其他格式。

### 2.2 基础语法
#### 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```

#### 段落与换行
段落之间用空行分隔。行末加两个空格可强制换行。

#### 强调
```markdown
*斜体* 或 _斜体_
**粗体** 或 __粗体__
***粗斜体***
```

#### 列表
无序列表使用`-`、`+`或`*`：
```markdown
- 项目1
- 项目2
  - 嵌套项目（缩进2或4空格）
```
有序列表使用数字加点：
```markdown
1. 第一项
2. 第二项
```

#### 链接与图片
```markdown
[链接文本](https://example.com)
![图片描述](图片地址)
```

#### 引用
```markdown
> 这是一段引用。
> 可以跨多行。
```

#### 代码
- 行内代码：用反引号包裹，如 `code`。
- 代码块：用三个反引号包裹，并可指定语言：
```markdown

```python
print("Hello")
```

```

#### 分割线
三个或更多`-`、`*`或`_`。

#### 转义字符
用反斜杠`\`转义Markdown符号。

### 2.3 扩展语法（GFM）
GitHub Flavored Markdown（GFM）增加了更多特性：
- **任务列表**：

```markdown
- [x] 已完成任务
- [ ] 未完成任务
```

- **表格**：

```markdown
| 姓名 | 年龄 |
|------|------|
| 张三 | 20   |
| 李四 | 22   |
```

- **删除线**：`~~删除线~~`
- **表情符号**：`:smile:` 等（GitHub支持）
- **自动链接**：直接粘贴URL会被自动识别为链接。
- **围栏代码块**：已支持，并可指定语言。

## 第三部分：LaTeX教程

### 3.1 LaTeX简介
LaTeX是一种基于TeX的排版系统，由Leslie Lamport开发。它通过标记来描述文档结构，让作者专注于内容，排版由系统自动处理。LaTeX在学术界（尤其是数学、物理、计算机科学领域）广泛使用，因为其强大的数学公式排版能力和对大型文档（如学位论文、书籍）的优秀支持。

### 3.2 文档结构与编译
一个最简单的LaTeX文档：
```latex
\documentclass{article}  % 文档类型：article, book, report等
\usepackage[UTF8]{ctex}  % 支持中文（使用ctex宏包）
\begin{document}
\title{我的第一个LaTeX文档}
\author{作者}
\date{\today}
\maketitle

Hello World!
\end{document}
```
编译命令（使用pdflatex或xelatex）：
```bash
xelatex mydoc.tex
```

### 3.3 文字与段落排版
#### 字体样式
```latex
\textbf{粗体}
\textit{斜体}
\underline{下划线}
\texttt{等宽字体}
```
#### 字体大小
```latex
{\tiny 极小}
{\small 小号}
{\large 大号}
{\huge 巨大}
```
#### 段落与换行
- 空行分隔段落。
- `\\` 强制换行（但不建议，可用空行分段）。
- `\par` 新段落。

#### 列表
```latex
% 无序列表
\begin{itemize}
    \item 项目1
    \item 项目2
\end{itemize}

% 有序列表
\begin{enumerate}
    \item 第一项
    \item 第二项
\end{enumerate}

% 描述列表
\begin{description}
    \item[术语] 解释
\end{description}
```

### 3.4 数学公式
LaTeX最强大的功能之一。数学模式分为行内和行间。

#### 行内公式
用`$...$`包裹：
```latex
爱因斯坦的质能方程 $E=mc^2$ 非常著名。
```

#### 行间公式
- 单行：用`\[...\]`或`$$...$$`（后者不推荐）。
```latex
\[
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
\]
```
- 多行对齐：`align`环境（需`amsmath`宏包）：
```latex
\usepackage{amsmath}
\begin{align}
    (x+y)^2 &= x^2 + 2xy + y^2 \\
    (x-y)^2 &= x^2 - 2xy + y^2
\end{align}
```

#### 常用数学符号
- 上下标：`x^2`、`a_i`、`e^{i\theta}`。
- 分式：`\frac{分子}{分母}`。
- 根式：`\sqrt[n]{表达式}`。
- 求和、积分：`\sum_{i=1}^{n}`、`\int_{a}^{b}`。
- 矩阵：`\begin{matrix} ... \end{matrix}`（需用括号包裹）。
- 希腊字母：`\alpha`、`\beta`、`\gamma`等。

### 3.5 表格与图形
#### 表格
```latex
\begin{tabular}{|c|c|}
    \hline
    姓名 & 年龄 \\
    \hline
    张三 & 20 \\
    李四 & 22 \\
    \hline
\end{tabular}
```
参数：`l`左对齐，`c`居中，`r`右对齐，`|`竖线，`\hline`水平线。

#### 插入图片
需加载`graphicx`宏包：
```latex
\usepackage{graphicx}
...
\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.5\textwidth]{image.png}
    \caption{图片标题}
    \label{fig:myimage}
\end{figure}
```

### 3.6 参考文献与交叉引用
#### 交叉引用
```latex
如图\ref{fig:myimage}所示...
```
需编译两次才能正确解析引用。

#### 参考文献
使用BibTeX：
- 创建`.bib`文件：
```bibtex
@book{lamport1994latex,
  title={LaTeX: A Document Preparation System},
  author={Lamport, Leslie},
  year={1994},
  publisher={Addison-Wesley}
}
```
- 在文档中引用：
```latex
\cite{lamport1994latex}
\bibliographystyle{plain}
\bibliography{refs}  % refs.bib
```
需用`latex -> bibtex -> latex -> latex`编译。

## 第四部分：综合应用

### 4.1 Markdown中嵌入HTML
Markdown兼容HTML，可以直接在Markdown文档中使用HTML标签来实现更复杂的布局。例如：
```markdown
# 标题

<div style="color: red;">
这部分文字是红色的，使用HTML实现。
</div>

普通Markdown段落。
```
在渲染时，HTML部分会被保留。

### 4.2 Markdown中嵌入LaTeX数学公式
许多Markdown渲染器（如GitHub、Typora、Jupyter Notebook）支持通过MathJax或KaTeX渲染LaTeX数学公式。语法与LaTeX类似：
- 行内公式：`$E=mc^2$`
- 行间公式：`$$E=mc^2$$` 或 `\[ E=mc^2 \]`

例如：
```markdown
质能方程 $E=mc^2$ 是著名的公式。

牛顿-莱布尼茨公式：
$$
\int_a^b f(x) dx = F(b) - F(a)
$$
```

### 4.3 从Markdown生成HTML/PDF
使用**Pandoc**可以轻松将Markdown转换为多种格式：
```bash
# Markdown转HTML
pandoc input.md -o output.html

# Markdown转PDF（需安装LaTeX）
pandoc input.md -o output.pdf --pdf-engine=xelatex

# 带LaTeX数学的Markdown转PDF
pandoc input.md -o output.pdf --pdf-engine=xelatex -V geometry:margin=1in
```

### 4.4 在HTML中使用LaTeX（MathJax）
MathJax是一个JavaScript库，可以在网页中渲染LaTeX数学公式。只需在HTML中引入MathJax，并用`$...$`或`$$...$$`书写公式即可。
```html
<!DOCTYPE html>
<html>
<head>
    <script src="https://polyfill.io/v3/polyfill.min.js?features=es6"></script>
    <script id="MathJax-script" async src="https://cdn.jsdelivr.net/npm/mathjax@3/es5/tex-mml-chtml.js"></script>
</head>
<body>
    <p>质能方程：\( E = mc^2 \)</p>
    <p>积分公式：\[ \int_a^b f(x)dx \]</p>
</body>
</html>
```
